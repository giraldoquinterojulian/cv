# CV dinámico de Julián Giraldo

Este repositorio contiene mi **CV interactivo en HTML**, pensado para compartirse como enlace público y consultarse desde navegador en versión visual o ATS.

## Acceso online
Una vez activado GitHub Pages, el CV estará disponible en:

`https://giraldoquinterojulian.github.io/cv/`

## Contenido del repositorio
- `index.html` → versión principal del CV interactivo

## Características
- Visualización web responsive
- Versión en español e inglés
- Modos de visualización por enfoque profesional
- Modo ATS
- Exportación a PDF desde navegador
- Diseño autocontenido en un único archivo HTML

## Perfiles objetivo
El CV está orientado a posiciones como:
- IT Project Manager
- IT Business Partner
- PMO
- Change Management
- Digital Transformation

## Publicación con GitHub Pages
Para publicar este CV:

1. Subir el archivo `index.html` al repositorio
2. Ir a **Settings**
3. Entrar en **Pages**
4. En **Build and deployment**, seleccionar:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Guardar los cambios

GitHub generará una URL pública para compartir en LinkedIn, CV PDF o email.
